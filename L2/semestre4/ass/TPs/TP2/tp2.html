1 Écrire sur la sortie standard

Vous allez enfin pouvoir afficher des informations sur la sortie standard :)

Pour cela, il faut :

  • invoquer la fonction système pour l'écriture :

movl $4, %eax

  • préciser que l'affichage se fait sur la sortie standard :

movl $1, %ebx

  • donner l'adresse de l'espace mémoire contenant les données à écrire

movl $tab, %ecx

  • donner le nombre d'octets à écrire

movl nb_car, %edx

  • lancer une interruption système pour exécuter l'instruction

int $0x80

Les registres sont obligatoirement ceux qui sont indiqués (numéro de la fonction dans %eax, adresse de la donnée dans %ecx etc etc).

Voici un exemple qui affiche un message puis des caractères. 

	.data
bonjour:
	.ascii	"Bonjour ! "
etoile:
	.byte	'*'
nl:	.byte	'\n'
	.text
	.globl	main
main:	# afficher la chaîne de caractères
	movl	$4, %eax	# fonction sys_write 
	movl	$1, %ebx	# sur la sortie standard (écran)
	movl	$bonjour, %ecx	# adresse du premier car. à écrire
	movl	$10, %edx	# nombre de caractères
	int	$0x80		# lancement de l'interruption

	#  afficher l'étoile
	movl	$4, %eax	
	movl	$1, %ebx	
	movl	$etoile, %ecx	
	movl	$1, %edx	
	int	$0x80

	# aller à la ligne
	movl	$4, %eax	
	movl	$1, %ebx	
	movl	$nl, %ecx	
	movl	$1, %edx	
	int	$0x80

Copiez-le, testez-le.

2.3 Avec des opérations logiques

Exercice

Ecrivez un programme qui affiche une valeur codée sur 8 octets. On prendra les conventions suivantes : un octet est affiché sur une ligne, un bit est représenté par un espace si le bit est à 0, par une étoile si le bit est à 1.

Par exemple, si la valeur est 0xAB34CD0145D43E4A ce qui revient, en binaire (les espaces sont ajoutés pour la compréhension), à 10101011 00110100 11001101 00000001 01000101 11010100 00111110 01001010, voilà ce qui doit être affiché : 

* * * **
  ** *
**  ** *
       *
 *   * *
** * *
  *****
 *  * *

Vous pourrez utiliser les valeurs suivantes pour tester votre programme :

zero: .quad 0x18242424242418

un: .quad 0x0828480808081C

deux: .quad 0x1824040810203C
